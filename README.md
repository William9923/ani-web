# ani-web

Browser-based anime streaming frontend for AllAnime.

## Quick Start

```bash
npm install
npm start
```

Open http://localhost:9001 in your browser.

## Deployment

Deploy to Vercel:

```bash
npm i -g vercel
vercel
```

## Features

- Search anime by title
- Browse episodes (subbed/dubbed)
- Stream video with quality selection
- Watch history saved locally
